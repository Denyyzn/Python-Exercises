005 - Maximum

Easy

Description

Create a function called maximum that receives three numbers and returns the largest one.

Example

maximum(3, 7, 5)
# -> 7

maximum(10, 2, 8)
# -> 10

Test

To test your solution, run: 

python maximum.py

Three numbers enter. Only one leaves with the title of biggest.
