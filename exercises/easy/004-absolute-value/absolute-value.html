004 - Absolute Value

Easy

Description

Create a function called absolute_value that receives a number and returns its absolute value.

Example

absolute_value(10)
# -> 10

absolute_value(-7)
# -> 7

Test

To test your solution, run: 

python absolute-value.py

Negative numbers are not invited to stay negative here. The function politely flips them around.
