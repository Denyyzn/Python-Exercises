002 - Is Even

Easy

Description

Create a function called is_even that receives a number and returns True if the number is even and False if it is odd.

Example

is_even(4)
# -> True

is_even(7)
# -> False

Test

To test your solution, run: 

python is-even.py

If any test fails, your function still needs some work. The tests will show exactly which input didn't behave as expected.
