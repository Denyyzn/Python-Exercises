Logic Challenges

A collection of programming challenges made to practice logic using Python.

Difficulty Levels

  • Easy
  • Medium
  • Hard

Each Challenge Contains

  • A short description of the problem
  • A starter file
  • Automated tests
  • A solution in the resolutions/ folder

Repository Structure

exercises/
├── easy/
├── medium/
└── hard/

Each challenge follows this structure:

000-example/
├── example.md
├── example.py
├── resolutions/
│   ├── example.res00.py
│   └── example.res01.py
└── tests/
    └── tests.py

How To Use

  1. Choose a challenge
  2. Read the challenge file
  3. Open the .py file
  4. Write your solution
  5. Run the file to execute the tests
python challenge.py

If all tests pass, your solution is correct.

Difficulties

  • Start here: Easy
  • Then continue to: Medium
  • Finally: Hard

Notes

  • Try to solve the challenge before looking at the solution
  • The solutions are always inside the resolutions/ folder
  • Some problems may have more than one valid solution

Good luck and have fun.
